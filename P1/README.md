# Practica 0

Para ejecutar el notebook

1. Encontrarse dentro de la carpeta de la P1
``` shell
pwd
# Output: some_path/P1
```

2. Crear un entorno virtual.
``` shell
python -m venv venv
# si no funciona usar
python3 -m venv venv
```

3. Activar el entorno virtual.
``` shell
# Windows
. .\venv\Scripts\activate

# Linux
source /venv/bin/activate
```
